# Fitness Tracker

PWA-приложение для учёта физической активности и анализа InBody.

См. [`specs/`](specs/) и [`docs/architecture/`](docs/architecture/).

## Dev

```bash
uv sync                 # установить зависимости
uv run pytest           # тесты
uv run ruff check .     # линтер
uv run mypy             # типы
```
