# Fitness Tracker — Specifications Index

Web-мобильное приложение (PWA) для учёта физической активности, анализа InBody-показателей и автоматического формирования персонализированных планов тренировок.

**Команда:** Егор (AI-генератор тренировок), Маша (AI-прогноз InBody, рекомендации по адаптации плана).

**Стек (зафиксирован):**
- Backend: FastAPI, Python, PostgreSQL, Docker, nginx, uv
- Frontend: Flutter, Dart, shadcn (Flutter port), Riverpod, Dio, http_mock_adapter
- ML: Python (PyTorch/Scikit-Learn) — две собственные модели: workout generator + InBody predictor

## Открытые вопросы (требуют решения до начала разработки)

1. **Видео-демонстрация упражнений** — будет ли хост видео или только текст + статичные картинки?
2. **Персональные тренеры/фитнес-клубы** в ЦА — в MVP делаем только B2C (один пользователь = один аккаунт), B2B-кабинет тренера откладываем.
3. **Локали** — MVP только на русском или нужен EN сразу?
4. **GDPR-процедуры** (export/delete account) — описаны в 002, но юридический текст согласия пишет не команда.
